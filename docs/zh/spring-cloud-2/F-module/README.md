---

- **服务发现**
    - **提供 HTTP 和 DNS 两种发现方式**
- **健康监测**
    - **支持多种协议，HTTP、TCP、Docker、Shell脚本定制化**
- **KV 存储**
    - **key , Value 的存储方式**
- **多数据中心**
    - **Consul 支持多数据中心**
- **可视化 Web 界面**
