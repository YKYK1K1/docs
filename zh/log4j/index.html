# Log4j简介

# 概述

Log4j简介一个完整的软件,日志是必不可少的。程序从开发、测试、维护、运行等环节,都需要向控制台或文件等位置输出大量信息。这·些信息的输出,在很多时候是使用system.out.printIn()(影响系统运行效率，关闭比不关闭效率快十倍)无法完成的。

日志信息根据用途与记录内容的不同,分为调试日志、运行日志、异常日志等。

Log4j的全称为Log for java ,即专门用于Java语言的日志记录工具。
