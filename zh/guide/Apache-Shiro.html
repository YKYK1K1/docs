# Apache Shiro

Apache Shiro是一个开源安全框架，提供身份验证、授权、密码学和会话管理。Shiro框架直观、易用，同时也能提供健壮的安全性。

Shiro是一个强大、易用的Java安全框架。被用作于认证，授权，加密，session管理，依赖于Shiro简单易懂的API，可以快速的构建包括手机，大型web和商业应用。
